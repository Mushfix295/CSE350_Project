# CSE350_Project
Digital Electronics and Pulse Techniques

The Automated Rail Gate System is a digital logic-based automation project designed to enhance safety at railway crossings by eliminating manual intervention. The system utilizes three IR sensors strategically placed to detect a train's position—whether it is approaching, on the track, or passing. By integrating these sensors with AND and NOT gates, the circuit implements a fail-safe binary logic to control gate status.
​

Project Overview
The core objective of this project is to reduce human error and improve traffic flow through low-cost, reliable hardware integration. This system effectively simulates a real-world safety mechanism by triggering a Red LED "gate closed" signal whenever a train is detected.
​

Uses a combination of IR sensors and basic logic gates (AND, NOT) for automated decision-making.
​

Operates on a binary logic where any sensor trigger results in a low output from the AND gate, subsequently inverted by the NOT gate to power the indicator.
​

Addresses environmental challenges, such as sensor calibration for ambient light, through physical casing and circuit adjustments.
​

Features a simplified hardware stack including transistors, diodes, and resistors on a breadboard setup.
​

Demonstrates reliable detection of movement and immediate status response.
​

Built as a foundation for future enhancements like servo-motor gate control and GSM-based alert systems.
​

Provides a cost-effective alternative to manual gate operation in high-risk regions.
​

Highlights practical application of digital logic design in solving real-world safety issues.
